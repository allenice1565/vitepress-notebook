```typescript
enum DirectionEnum {
	left: 0,
	right: 1,
	top: 2,
	bottom: 3,
}
```

获取`DirectionEnum`的key对应的类型：

```typescript
const keyType = keyof typeof DirectionEnum; // 'left' | 'right' | 'top' | 'bottom'
```

获取`DirectionEnum`的value对应的类型：

```typescript
const valueType = `${DirectionEnum}`; // 0 | 1 | 2 | 3
```
